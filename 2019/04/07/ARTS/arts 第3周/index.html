146.LRU Cache

1007.Minimum Domino Rotations For Equal Row
两个列表元素交换次数最少,达到其中一个列表元素全部相同.
思路:贪心,要全部相同,必定其中一个值得次数超过1/2.
坑是:条件给了两列表相等,实际有一组数据两列表不等..
