algorithm

https://leetcode.com/problems/longest-substring-without-repeating-characters/
没有重复的最长子串.注意,子串,有连续性的要求
思路,简单动归.
输入s字符串
f[I] 以第i位结尾的没有重复的最长子串,包含i位
a[s[i]] s[i]上一次出现的位置,第一次出现,a[s[i]] = -1
动归方程式:
f[i] =
f[i-1]+1,s[i]第一次出现
min(i-a[s[i]],f[i-1]+1),s[i]不是第一次出现

解释:
如果s[i]第一次出现,那么问题就转化为前i-1中,因为s[i]是第一次出现,i-1必然和i位不同,所以问题就转化为f[i-1]+1

min(i-a[k],f[i-1]+1)
两个边界:
1.f[I-1]+1应该是最大值,因为再往前,f[i-1]的条件就不满足的
2.而i-a[s[i]],则是f[i]本身的边间,否则就出现s[I]重复

https://leetcode.com/problems/median-of-two-sorted-arrays/
这是一道水题,
两个有序列表求中位数.
思路很简单,先合并,再求中位数
合并的方式不是先随意合并,再排序,这样时间复杂度就变为了
O((n+m)log2(n+m)).
两列表有序,类似归并排序的合并方式,只需要O(n+m)时间
